Mixtures of Probabilistic Principal Component Analysis (MPPCA)

Usage Guide:

In order to use MPPCA on your data, you need to:

  1. Specify the model parameters in the sidebar on the left
  2. Click the 'SUBMIT PARAMETERS' button for the model to run.
  3. Once the model finishes running, plots will appear under the different plot tabs beside the 'Description' tab.

There are a few parameters which can be tune according to interest:

  1. Range of Groups Estimated (e.g: 1 to 4) - XXXXXXXXXXXXXXXXXXXXXX
  2. Range of Principal Components (e.g: 1 to 4) - This will run 4 MPPCA models with 1, 2, 3, and 4 PC scores in the model.
  3. Number of Bootstrap - This quantifies the uncertainty level of the model parameters using the Bootstrap method. Higher number will increase the accuracy of the given uncertainty but at the cost of higher computational running time. The minimum bootstrap run is according to the number of rows in the data.
  4. Convergence Criterion - The value where it decides that the model has converged. Smaller number will have longer computational time but better accuracy.
  5. Maximum Number of Iteration - The maximum number of iterations the model runs the EM algorithm before it is terminated regardless of the results.
