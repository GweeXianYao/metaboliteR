Probabilistic Principal Component and Covariates Analysis (PPCCA)

Parameters Description:

There are a few parameters which can be tune according to interest:

  1. Range of Principal Components (e.g: 1 to 4) - This will run 4 PPCA/PPCCA models with 1, 2, 3, and 4 PC scores in the model.
  2. Number of Bootstrap - This quantifies the uncertainty level of the model parameters using the Bootstrap method. Higher number will increase the accuracy of the given uncertainty but at the cost of higher computational running time. The minimum bootstrap run is according to the number of rows in the data
  3. Include Covariates - This will run the PPCCA model which will account for the given covariates data. This is disabled when there is no covariates data given in the 'Data' tab.
  4. Output Optimize Model - This will decide which PPCA/PPCCA model will be shown in the output. Optimal will be chosen based on statistical elbow of BIC. Maximum will just choose the PPCA/PPCCA model with highest number of principal components.
  5. Convergence Criterion - The value where it decides that the model has converged. Smaller number will have longer computational time but better accuracy.
  6. Maximum Number of Iteration - The maximum number of iterations the model runs the EM algorithm before it is terminated regardless of the results.
