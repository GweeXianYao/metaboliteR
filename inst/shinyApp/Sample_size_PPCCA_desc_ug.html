Sample Size Estimation using PPCCA Analysis

Usage Guide:

In order to estimate Sample Size using PPCA on your data, you need to:

  1. Specify the scaling type in the sidebar on the left
  2. After selecting the scaling type, you can now specify the model parameters and click the 'calculate' button and verify the entered model parameters in 'Values' tab besides 'Descrition' tab..
  3. Once the model finishes running, plots for "Sample Size Estimation"" and "Varying the Significant bins"" will appear besides 'Values' tab.

There are a few parameters which can be tune according to interest:

  1. Spectral Bins (i.e. 1 to 1000) -
  2. Proportion of significant bins (i.e. 0.1 to 1) - Expected proportion 'm' of significant metabolites.
  3. Target FDR (i.e. 0.01 to 0.2)- Desired level of FDR.
  4. To investigate the impact of 'proportion of significant bins'(m) on FDR:
    • Sample size for group 'n1'
    • Sample size for group 'n2'
