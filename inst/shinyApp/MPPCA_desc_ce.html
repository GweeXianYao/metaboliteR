Mixture of Probabilistic Principal Component Analysis (MPPCA)

MPPCA is an extension of the PPCA method which offers group clustering benefit. As observation in a data set can have underlying group structure, MPPCA attempts to identify these underlying groups by using a mixture of PPCA models and calculate the model parameters in a two stage EM algorithm called the Alternating Expectation Conditional Maximization (AECM) algorithm.
